Security & Hosting
Azure Government Cloud
All ReUse360™ data is hosted on FedRAMP-aligned Azure Government infrastructure. US soil data residency. TLS 1.3 encryption in transit, AES-256 at rest. NIST 800-53 Moderate target baseline.

✓ County retains ownership of all customer data

⚠ Integration Confirmations Required
  • AMI TEAM / BADGER METER — Confirm Beacon REST API availability, authentication method, and data dictionary for interval read access by third-party SaaS vendor.
  • GIS / ENTERPRISE TECHNOLOGY — Confirm availability of parcel feature service (GeoJSON or REST) including meter ID, zoning, and HOA boundary fields for vendor integration.
  • SHANE (CITYWORKS ADMIN) — Confirm Cityworks REST API supports external SR creation. Provide irrigation violation SR template ID and required payload fields.
  • IT / FINANCE SYSTEMS — Confirm SAP read-only integration method (API or scheduled extract) for customer account, meter ID, and rate code fields.
